Projects

Travel Hindustan

A tourism website for users to explore various tourist destinations in India. Users were provided with hotel options and directed to the booking websites. News, blogs, reviews were also provided for the destinations. They could also see upcoming events and festivals.

Todo-List

A todo-list where user can add any number of todos and delete todos. Description can also be added. Local storage is used to save the todos so that they are not erased when the website is not running. Validation is done so no empty todo can be added.

Phone Book

A Phone book where user can save the contact details, delete and edit them as well. A search box is also provided which shows the saved contacts and user can search contact by name. Database was used to save the data and sqlite was connected with python and tkinter.

A blogging website for the students of university. They can share their views and technical informations by writting blogs. Pictures can also be be added. Official announcements and articles can also be done by admin for interaction with all the students.

Digit Prediction

A Machine Learning project using the mnist dataset of handwritten digits. SGD classifier was used to predict if the given number is correctly predicted as in the target. Accuracy of correct prediction of handwritten digits from 0 to 9 was calculated.
